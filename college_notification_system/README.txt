
IIT Chennai Notification System - Final (PHP + MySQL)

Installation:
1. Copy folder to XAMPP htdocs (e.g., C:\xampp\htdocs\iit_notification_system_final)
2. Start Apache & MySQL.
3. Open phpMyAdmin and import install.sql to create database and tables.
4. Edit config/db.php if your MySQL user/password differs.
5. Visit http://localhost/iit_notification_system_final/
6. Register users or use provided VC sample (password may need update).

Features:
- Bootstrap UI with sidebar
- Dynamic registration (Principal/HOD/Student specifics)
- Send notifications with attachments (PDF, images)
- Fine-grained targeting for students (college→branch→year)
